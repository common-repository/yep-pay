== Description ==

Yep! Pay is a Nigerian online payment gateway which offers an Innovative, Dependable, Easy-to-use, Affordable and Secure payment solution
Yep! Pay allows site owners to receive payments for goods and services purchased on their website.
Before you can start accepting payment on your website, login to your Yep! Pay accountto retrieve your API Keys
If you do not have a CashEnvoy Business Account Sign up here. (https://yeppay.io/register) with no setup Fee.
Yep! PayWoocommerce Payment Gateway allows you to accept payment on your Woocommerce store using Nigeria issued Visa Card, Mastercard, Verve Cards.
With Yep! PayWoocommerce Payment Gateway plugin, you will be able to accept the following payment methods in your shop:
* __MasterCards__
* __Visa Card__
* __Verve Cards__
* __USSD__
* __Transfer__


= Note =

This plugin is suitable for only merchants in Nigeria.


= Plugin Features =

* __Accept payment__ via Visa Card, Mastercard, Verve Card.
* __Seamless integration into the WooCommerce checkout page.
* __Add Naira__ currency symbol. To select it go to go to __WooCommerce__ > __Settings__ from the left hand menu, then click __General__ from the top tab. From __Currency__ select Naira, then click on __Save Changes__ for your changes to be effected.

== Installation ==

*   Go to __WordPress Admin__ > __Plugins__ > __Add New__ from the left-hand menu
*   In the search box type __Yep Pay__
*   Click on Install now when you see __Yep Pay__ to install the plugin
*   After installation, __activate__ the plugin.
